# ☣️ 荒疫（Blight）

## ✦ 简介
`荒疫（Blight）` 是一个面向 **Slay the Spire 2** 的高难度挑战 Mod。

它以原版 **A10** 作为基础，围绕 **卡牌附魔 / 锻造** 构筑出一套独立的挑战体系：**荒疫进阶**、**变异敌人**、**双波战斗**、**开局负面效果** 与 **Boss 特殊强化**。

这不是对原版难度的轻微上调，而是一个强调 **高风险 / 高回报 / 高构筑上限** 的新模式。

---

## ✨ 你可以期待什么

- **独立模式入口**：主菜单新增 `荒疫模式` 与 `荒疫联机` 按钮
- **独立难度曲线**：不复用原版 Ascension 文案，采用单独的 0–5 级荒疫进阶
- **更危险的战斗**：变异敌人、双波战斗与强化 Boss 会持续提高战斗压力
- **更夸张的成长**：通过战斗获取锻造奖励，为卡牌附加强力或带代价的特殊效果
- **更完整的模式识别**：角色选择、局内提示与相关 UI 会显示荒疫模式对应文本

---

## 📈 荒疫进阶

荒疫模式当前拥有独立的 **0–5 级难度**：

- **荒疫 0**：以原版 A10 作为基础难度
- **荒疫 1**：普通战和精英战中有概率出现 **变异敌人**
- **荒疫 2**：精英敌人更危险，并获得额外随机强化
- **荒疫 3**：开局获得一个随机负面效果
- **荒疫 4**：地图中有概率出现 **双波敌人战斗**
- **荒疫 5**：Boss 获得额外的荒疫能力

> 当前版本仍为 Beta。现阶段部分敌人的荒疫化内容仍以血量变化为主，更多 AI 与特殊能力会在后续版本继续补完。

---

## ⚔️ 核心玩法亮点

### 🧬 变异敌人
从荒疫 1 开始，部分战斗会出现变异个体。

它们不只是数值更高，而是会改变战斗预期、奖励价值与路线决策，让每一局都更不稳定，也更有压迫感。

### ☠️ 开局负面效果
随着荒疫等级提升，你可能会在开局就背负负面效果。

这会直接影响前期生存、资源管理与整局节奏，让路线规划从第一场战斗前就开始变得重要。

### 🌊 双波敌人战斗
高难度荒疫下，部分战斗会以双波形式出现。

你需要考虑的不再只是“当前这一波怎么打”，还包括续航、保留资源，以及如何处理下一波敌人。

### 🔥 卡牌附魔与锻造奖励
荒疫模式加入了独立的 **附魔 / 锻造奖励** 体系。

在精英战、变异战等高风险战斗后，你有机会获得特殊锻造奖励，为卡牌赋予额外效果，例如：

- 额外伤害 / 额外格挡
- 费用降低
- 保留 / 固有 / 永恒 / 重放等关键词增益
- 带有明显代价的诅咒型强化
- 偏成长、连打或爆发向的特殊效果

这让荒疫模式不仅更难，也拥有更高的构筑爆发力和成长空间。

# ☣️ Blight

## ✦ Overview
`Blight` is a high-difficulty challenge mod for **Slay the Spire 2**.

Built on top of vanilla **A10**, it introduces a separate progression system centered on **card enchantment / forging**, combined with **Blight Ascension**, **mutant enemies**, **double-wave battles**, **starting debuffs**, and **boss enhancements**.

This is not a small vanilla difficulty bump. It is a standalone challenge mode built around **high risk, high reward, and high build potential**.

---

## ✨ What This Mod Adds

- **Separate mode entry**: New `Blight Mode` and `Blight Multiplayer` buttons on the main menu
- **Separate difficulty curve**: A dedicated 0–5 Blight Ascension system instead of vanilla Ascension text
- **More dangerous combat**: Mutant enemies, double-wave encounters, and upgraded bosses
- **Bigger build payoff**: Forge rewards that add powerful or risky extra effects to cards
- **Better mode clarity**: Character select and in-run UI display Blight-specific text and tooltips

---

## 📈 Blight Ascension

Blight Mode currently uses its own **0–5 difficulty scale**:

- **Blight 0**: Uses vanilla A10 as the baseline difficulty
- **Blight 1**: **Mutant enemies** can appear in normal and elite fights
- **Blight 2**: Elites become more dangerous and gain extra random upgrades
- **Blight 3**: Start each run with a random negative effect
- **Blight 4**: **Double-wave battles** can appear on the map
- **Blight 5**: Bosses gain additional Blight abilities

> This mod is still in Beta. At the moment, some enemies mainly use HP-based scaling, while more custom AI and special mechanics are planned for future updates.

---

## ⚔️ Core Highlights

### 🧬 Mutant Enemies
Starting from Blight 1, some encounters can spawn mutant variants.

These fights are not just simple stat inflation. They change combat expectations, reward value, and route planning, making runs more volatile and more demanding.

### ☠️ Starting Debuffs
At higher Blight levels, each run may begin with a random negative effect.

That pressure starts immediately, affecting early survival, resource management, and long-term planning.

### 🌊 Double-Wave Battles
Some high-difficulty encounters can appear as double-wave fights.

You need to plan for more than the current enemy group: sustain, resource preservation, and second-wave handling all become important.

### 🔥 Card Enchantments and Forge Rewards
Blight Mode includes a dedicated **enchantment / forge reward** system.

After high-risk fights such as elite battles and mutant encounters, you may receive forge rewards that add extra effects to cards, including:

- Extra damage / extra block
- Cost reduction
- Keyword bonuses such as Retain / Innate / Eternal / Replay
- Curse-like upgrades with meaningful drawbacks
- Scaling, combo, or burst-oriented special effects

The result is a mode that is not only harder, but also capable of much more extreme builds.

---

## 🎯 Who Is It For?

Recommended for players who:

- Already know the vanilla game well and want a harsher challenge
- Enjoy risk/reward systems
- Like build variety, card enhancement, and random power growth
- Want a progression curve different from vanilla Ascension

If you prefer a tightly controlled experience close to vanilla balance, this mod may feel too aggressive.
