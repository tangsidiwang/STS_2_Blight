# 荒疫模式 - 新增内容清单草案 (Content Additions)

## 1. 全新遗物设计预案 (New Relics)
> *"需要添加新的遗物，卡牌，附魔。"*

本组件将在荒疫模式被解锁时或者仅限该模式下生效：

*   **荒疫之核 (Blight Core)** (稀有/Boss)：初始携带。当你对卡牌附加第3个附魔时，那次附魔不会消耗金钱或资源，且这局游戏中该卡牌获得一次免消耗（如果符合的话）。
*   **腐化的铁砧 (Corrupted Anvil)** (罕见)：在战斗结束后，有 10% 的几率使掉落的金币直接转化为一个“锻造奖励”（Forge Reward）。
*   **多面棱镜 (Prism Fragment)** (普通)：带有附魔的卡牌打出时，额外获得 2 点临时格挡。
*   **附魔吸收器 (Enchantment Vacuum)** (稀有)：每当将带有多重附魔的牌消耗（Exhaust）时，随机对随机单体敌人造成当前附魔数量 × 5 的伤害。

## 2. 全新卡牌设计预案 (New Cards)
专门为了互动“多重附魔机制”而增加的特殊牌印：

*   **【附魔共鸣 (Resonance Enchant)】**
    *   费用：1
    *   类型：技能
    *   效果：选择手中一张牌。复制它身上所有的附魔效果，将它们转嫁给你本回合打出的下一张普通牌。
*   **【荒疫打击 (Blight Strike)】**
    *   费用：1
    *   类型：攻击
    *   效果：造成 8 点伤害。这张牌每拥有 1 个附魔，额外造成 4 点伤害。
*   **【极限萃取 (Extreme Extraction)】**
    *   费用：0
    *   类型：技能
    *   效果：消耗（Exhaust）本卡牌。本场战斗中，将抽牌堆最上面一张牌永久附加“锋利”附魔，但获得 1 张“伤口”。

## 3. 全新附魔清单草案 (Enchants)
需要基于前面 `02_Enchantment_System.md` 定义的规则池落地的实质选项：

| 附魔名称 | 池级 | 适用性 | 叠加行为 (再次获取时) |
| :--- | :--- | :--- | :--- |
| **锐利** | 普通 | 攻击 | 数值伤害 +3 -> +6 -> +9... |
| **坚固** | 普通 | 技能 | 数值格挡 +3 -> +6 -> +9... |
| **汲取** | 罕见 | 攻击 | 未阻挡伤害带来回血（比例放大或固定值叠加） |
| **轻盈** | 罕见 | 全牌 | 减少1点耗费，数值叠加最多减到0或保持上限0不可变。 |
| **双重** | 稀有 | 全牌 | （不叠加数值）赋予打出2次附带消耗的效果。如果获得两次，可能变为打出3次。 |
| **剧毒** | 罕见 | 攻击 | 每层附魔赋予攻击额外叠加 2 层毒。 |
| **保留** | 罕见 | 全牌 | （不叠加数值）回合结束时留在手中，下回合保留。 |
| **连锁** | 稀有 | 全牌 | 每获得一个，在使用此牌时就会自动额外打出一张基础“打击”或“防御”。 |

这些内容会在接下来的新对话中通过详细的C#类实现继承和参数调节。
